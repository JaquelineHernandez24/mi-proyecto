# 🌐 Manipulación de Escenarios 3D con Three.js

## 📚 Información General
- **Materia:** GRAFICACION
- **Tema:** Manipulación de escenarios 3D
- **Alumno:** HERNANDEZ HERNANDEZ JAQUELINE


---

## 🎯 Objetivo
Desarrollar una aplicación web que permita la manipulación de objetos 3D utilizando el mouse y teclado mediante la librería Three.js.

---

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- Three.js

---

## 🎮 Demos implementadas

Se integraron los siguientes ejemplos de Three.js:

- 🧱 Minecraft Geometry
- 🗺️ Map Controls
- 🌀 Orbit Controls
- 🎯 Pointer Lock
- 🔧 Transform Controls
